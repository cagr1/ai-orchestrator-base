# Landing Page de Empresa de Seguridad

## 1. Información General

- **Nombre del proyecto**: Landing Page - Seguridad Integral
- **Tipo**: Landing page informativo
- **Objetivo principal**: Convertir visitantes en clientes potenciales mediante demostración de servicios de seguridad profesional
- **Target**: Empresas y particulares que buscan servicios de seguridad

## 2. Secciones del Landing Page

### 2.1 Header (Encabezado)
- Logo de la empresa
- Menú de navegación: Inicio | Servicios | Nosotros | Contacto
- Botón CTA: "Solicitar Cotización"

### 2.2 Hero Section
- Título principal: "Protección Integral para lo que más valoras"
- Subtítulo explicativo del servicio principal
- Botón CTA primario: "Ver Servicios"
- Botón CTA secundario: "Contactar Ahora"
- Imagen representativa: Equipo de seguridad profesional

### 2.3 Servicios (Services Section)
- Título: "Nuestros Servicios"
- Grid de 6 servicios principales:
  1. Vigilancia 24/7
  2. Sistemas de Alarmas
  3. Control de Acceso
  4. Protección Personal
  5. Seguridad Corporativa
  6. Monitoreo Remoto

### 2.4 Por Qué Elegirnos (Why Us Section)
- Título: "¿Por Qué Elegirnos?"
- 4 beneficios clave con iconos:
  1. Experiencia certificada
  2. Tecnología de última generación
  3. Respuesta inmediata 24/7
  4. Precios competitivos

### 2.5 Testimonios
- Título: "Lo que dicen nuestros clientes"
- 3 testimonios de clientes satisfechos
- Nombre, empresa y foto (placeholder)

### 2.6call to Action Final
- Título: "¿Necesitas protección?"
- Subtítulo convidativo
- Botón grande: "Obtener Cotización Gratis"

### 2.7 Footer
- Información de contacto
- Redes sociales
- Enlaces rápidos
- Copyright

## 3. Componentes Requeridos

| Componente | Descripción |
|------------|-------------|
| Navbar | Sticky, responsive, con menú hamburguesa en móvil |
| Hero | Full-width, background image, texto centrado |
| ServiceCard | Icono, título, descripción, botón "Saber más" |
| BenefitCard | Icono, título, descripción corta |
| TestimonialCard | Foto, nombre, empresa, texto, estrellas |
| CTAButton | Botón principal con hover effects |
| ContactForm | Campos: nombre, email, teléfono, mensaje |
| Footer | 4 columnas: logo, enlaces, contacto, redes |

## 4. Especificaciones Técnicas

- **Framework recomendado**: Next.js o React
- **Estilos**: Tailwind CSS
- **Responsive**: Mobile First (320px, 768px, 1024px, 1440px)
- **Imágenes**: Formato WebP optimizado
- **Accesibilidad**: WCAG 2.1 nivel AA
